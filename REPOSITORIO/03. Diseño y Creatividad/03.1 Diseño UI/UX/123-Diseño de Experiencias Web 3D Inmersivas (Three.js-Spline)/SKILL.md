## Descripción
Esta habilidad permite integrar la tercera dimensión en la web de manera estratégica y fluida. Desde configuradores de producto interactivos hasta escenas ambientales inmersivas, se enfoca en equilibrar el impacto visual con el rendimiento técnico (WebGL). Utiliza herramientas modernas como Spline para iteraciones rápidas y React Three Fiber para integraciones robustas en aplicaciones web complejas.

## Cuándo usarla
- **Lanzamientos de Producto Premium**: Para destacar detalles físicos y texturas mediante visualización 3D.
- **Storytelling Interactivo**: Narrativas que responden al scroll del usuario mediante cambios de cámara y animación de objetos.
- **Portafolios de Vanguardia**: Para crear una primera impresión de alto nivel técnico y artístico.

## Requisitos
- Conocimientos básicos de modelado 3D (Blender o Spline).
- Entorno de desarrollo compatible con WebGL.
- Bibliotecas: `three.js`, `@react-three/fiber`, `@react-three/drei`.
