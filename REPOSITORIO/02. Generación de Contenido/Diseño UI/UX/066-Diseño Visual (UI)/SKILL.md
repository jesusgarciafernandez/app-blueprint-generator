## Descripción
Habilidad estratégica y operativa centrada en la arquitectura visual de capas de interacción para productos digitales (web, aplicaciones móviles, dashboards). El Diseño UI se encarga de traducir la estrategia de experiencia de usuario (UX) en una interfaz tangible, estética y funcional. Integra principios avanzados de jerarquía visual, teoría del color aplicada, tipografía digital, sistemas de espaciado y micro-interacciones para crear entornos que no solo resulten atractivos y "Premium", sino que también optimicen la carga cognitiva, refuercen la identidad de marca y garanticen una navegación intuitiva y libre de fricción en cualquier dispositivo.

## Cuándo usarla
Escenarios que activan esta skill:
- Al conceptualizar la identidad visual y la capa de interacción de una nueva plataforma digital o herramienta de software.
- Durante la creación, mantenimiento o expansión de un Sistema de Diseño (*Design System*) para garantizar la escalabilidad y coherencia visual del producto.
- En la fase de alta fidelidad del diseño, transformando wireframes estructurales en prototipos interactivos con acabados realistas y funcionales.
- Para asegurar que el producto cumple con los estándares internacionales de accesibilidad visual (WCAG), optimizando contrastes y legibilidad.
- Al buscar elevar la percepción de calidad de un producto mediante el refinamiento estético de componentes, estados visuales y transiciones fluidas.

## Requisitos
- Dominio experto de herramientas de diseño de interfaz y prototipado (ej: Figma, Adobe XD, Framer o Sketch).
- Comprensión profunda de los lenguajes de diseño nativos (Material Design de Google y Human Interface Guidelines de Apple).
- Metodología de diseño basado en componentes (*Atomic Design*) y dominio de variables/tokens de diseño.
- Conocimientos de grid systems, auto-layout y principios de diseño responsivo/adaptativo.
- Capacidad de comunicación técnica para el traspaso (*Handoff*) de activos y especificaciones al equipo de desarrollo front-end.
