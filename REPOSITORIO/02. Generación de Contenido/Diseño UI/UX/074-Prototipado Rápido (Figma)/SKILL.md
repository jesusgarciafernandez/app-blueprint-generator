## Descripción
Habilidad técnica de alto rendimiento para la creación ágil de simulaciones interactivas de productos digitales utilizando Figma. El Prototipado Rápido permite recrear la experiencia real de uso de una aplicación o sitio web (navegación, micro-interacciones, lógica condicional y animaciones complejas) sin incurrir en costes de desarrollo de software. Esta skill integra el uso avanzado de componentes atómicos, auto-layout, variables de diseño y motores de animación (*Smart Animate*) para validar hipótesis de diseño con stakeholders, realizar pruebas de usabilidad de alta fidelidad y garantizar un traspaso técnico impecable al equipo de ingeniería, detectando errores de flujo en etapas tempranas del ciclo de vida del producto.

## Cuándo usarla
Escenarios que activan esta skill:
- Al inicio de un ciclo de producto para presentar conceptos interactivos y obtener la aprobación de dirección o inversores con un modelo tangible.
- Para ejecutar sesiones de test de usabilidad (User Testing) con usuarios reales, observando su comportamiento ante un sistema que parece y responde como uno final.
- Durante la definición de micro-interacciones complejas (ej: drag & drop, estados de carga, scroll dinámico) que son difíciles de explicar solo con imágenes estáticas.
- Como herramienta fundamental de comunicación técnica para mostrar al equipo de desarrollo front-end el comportamiento exacto de las transiciones y estados de la interfaz.
- Para crear materiales de venta o demos de producto de alta fidelidad que puedan desplegarse en dispositivos móviles de forma inmediata (vía Figma Mirror).

## Requisitos
- Dominio avanzado del ecosistema Figma, específicamente en el manejo de Componentes, Variantes y Propiedades de Instancia.
- Dominio de la herramienta Auto-layout para crear interfaces flexibles que se adapten a diferentes resoluciones de pantalla.
- Acceso a las funcionalidades avanzadas de prototipado de Figma (Variables, Expresiones y Lógica Condicional) para simulaciones de alta fidelidad.
- Comprensión de los principios de diseño de interacción y leyes de respuesta visual (feedback dinámico).
- Biblioteca de componentes (System UI) pre-configurada para acelerar el proceso de ensamblado de pantallas.
