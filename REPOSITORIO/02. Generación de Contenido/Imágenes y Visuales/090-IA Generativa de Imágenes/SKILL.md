## Descripción
Habilidad creativa y técnica enfocada en el uso de modelos de difusión y transformadores visuales para generar imágenes, ilustraciones y prototipos visuales de alta calidad a partir de descripciones textuales o imágenes base. Esta skill abarca desde la redacción de prompts complejos hasta el control técnico fino mediante Inpainting, Outpainting, ControlNet y entrenamiento de LoRAs personalizados. La IA Generativa de Imágenes ha revolucionado el diseño gráfico, el marketing y la preproducción cinematográfica, permitiendo crear assets visuales únicos en segundos y democratizando la producción de contenido visual complejo.

## Cuándo usarla
Escenarios que activan esta skill:
- Durante la creación de maquetas (mockups) rápidas para el diseño de interfaces de usuario o productos físicos.
- Para generar imágenes de stock personalizadas que se alineen perfectamente con la identidad visual de una marca.
- Al producir contenido visual masivo para campañas de redes sociales sin depender de fotógrafos o ilustradores externos.
- En la fase de Concept Art para videojuegos o películas, explorando de forma iterativa estilos y personajes.
- Para restaurar fotos antiguas o modificar elementos específicos de una imagen (ej: cambiar el fondo de un producto).

## Requisitos
- Maestría en Prompt Engineering visual (uso de pesos, estilos, iluminación, cámaras).
- Dominio de plataformas como Midjourney, DALL-E 3 o interfaces locales (Automatic1111, ComfyUI).
- Conocimiento de técnicas de control espacial (ControlNet: Canny, Scribble, Depth).
- Capacidad para realizar retoques mediante Inpainting y escalado de alta resolución (Upscaling).
- Entendimiento de los aspectos éticos y legales del copyright en arte generado por IA.
