## Descripción
Esta habilidad es la "póliza de seguros" del desarrollo web de élite. El profesional de Web Testing diseña una pirámide de pruebas equilibrada que combina velocidad y confianza. Utiliza herramientas modernas (Playwright, Cypress, Jest, Vitest) para cubrir desde la lógica de negocio pura hasta los flujos de usuario completos (E2E) en múltiples navegadores. Se enfoca en la robustez del sistema, la detección temprana de regresiones y el aseguramiento de la accesibilidad (WCAG), permitiendo un desarrollo ágil sin miedo al fallo catastrófico. Es la base de la entrega continua de valor.

## Cuándo usarla
- **Desarrollo de Single Page Applications (SPA)**: Para asegurar que el complejo estado de la interfaz se mantiene coherente y funcional.
- **Procesos de Compra y Checkouts Críticos**: Donde un error en el flujo de usuario se traduce directamente en una pérdida irreversible de ingresos.
- **Plataformas de Alto Tráfico y Uso Masivo**: Para garantizar que las actualizaciones constantes no degraden la experiencia de usuario.

## Requisitos
- Herramientas de automatización de pruebas (Playwright, Jest, Testing Library, Cypress).
- Conocimientos de la Pirámide de Testing y patrones de diseño de pruebas (Page Object Model).
- Capacidad para integrar pruebas en entornos de CI/CD (GitHub Actions).
