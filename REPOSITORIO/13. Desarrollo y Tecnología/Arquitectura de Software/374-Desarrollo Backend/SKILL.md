## Descripción
Habilidad técnica fundamental enfocada en la construcción de la lógica de negocio, el procesamiento de datos y la comunicación con bases de datos en el lado del servidor. Esta skill abarca el desarrollo de sistemas robustos utilizando lenguajes de alto rendimiento (ej: Node.js, Go o Python), el diseño de esquemas de datos eficientes, la implementación de capas de seguridad (autenticación y autorización) y la orquestación de servicios asíncronos mediante colas de mensajes. El objetivo es proporcionar una base sólida, rápida y escalable que sustente las aplicaciones frontend y los servicios externos, garantizando la integridad de los datos y la eficiencia operacional.

## Cuándo usarla
Escenarios que activan esta skill:
- Al diseñar la API central de una plataforma que necesita centralizar la lógica de negocio y el acceso a datos.
- Durante el desarrollo de sistemas que requieren procesamiento intensivo de datos en segundo plano (Background Workers).
- Para implementar sistemas de autenticación segura (JWT, OAuth) y control de acceso basado en roles (RBAC).
- Al integrar servicios de terceros y sistemas de archivos complejos en aplicaciones empresariales.
- Para optimizar el rendimiento de una aplicación mediante el uso de cachés (Redis) y optimización de consultas SQL/NoSQL.

## Requisitos
- Dominio de al menos un lenguaje de backend (Node.js/TypeScript, Go o Python).
- Conocimientos sólidos de bases de datos relacionales (PostgreSQL/MySQL) y no relacionales (MongoDB).
- Entorno de ejecución y gestor de dependencias configurado.
- Comprensión de protocolos de comunicación (HTTP/HTTPS, WebSockets, gRPC).
- Experiencia básica en Docker para entornos de desarrollo aislados.
