## Descripción
Habilidad técnica avanzada para la creación de servicios de backend, APIs web y sistemas distribuidos utilizando el lenguaje Swift (Server-side Swift) y el framework Vapor. Esta skill se centra en el aprovechamiento de la seguridad de tipos, el alto rendimiento y la concurrencia nativa de Swift (Async/Await) para construir servidores robustos en entornos Linux y macOS. Abarca desde la gestión de bases de datos mediante Fluent ORM hasta la implementación de WebSockets y la integración con el ecosistema Apple, permitiendo una experiencia de desarrollo unificada (Full-stack Swift) y una arquitectura altamente eficiente en el consumo de recursos.

## Cuándo usarla
Escenarios que activan esta skill:
- Al desarrollar aplicaciones iOS que requieran un backend compartido compartiendo lógica de negocio, modelos de datos y validaciones estrechamente vinculadas.
- Para aprovechar el alto rendimiento y la seguridad de memoria de Swift en servicios que deban procesar grandes volúmenes de datos con baja latencia.
- Durante la migración de backends existentes hacia arquitecturas más seguras y fuertemente tipadas.
- Cuando se busca unificar el lenguaje de programación en todo el equipo de desarrollo (Mobile y Backend).
- Para implementar microservicios ligeros en contenedores Docker sobre infraestructuras cloud.

## Requisitos
- Dominio experto del lenguaje Swift (Structs, Protocols, Generics, Concurrency).
- Conocimientos del framework Vapor y su sistema de rutas.
- Experiencia en el uso de ORMs (Fluent) y bases de datos relacionales (PostgreSQL, MySQL).
- Familiaridad con el sistema de paquetes de Swift (SPM).
- Entendimiento de entornos Linux (Ubuntu) y contenedores para producción.
