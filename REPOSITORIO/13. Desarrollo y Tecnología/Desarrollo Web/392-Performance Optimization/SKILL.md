## Descripción
Habilidad técnica avanzada para maximizar la velocidad de carga, la interactividad y la estabilidad visual de aplicaciones web. Esta skill se centra en el dominio de las Core Web Vitals (LCP, FID, CLS) y el uso de herramientas de diagnóstico profundo como Chrome DevTools Performance Tab, Lighthouse y WebPageTest. Abarca técnicas de optimización tanto en el lado del servidor (compresión Gzip/Brotli, caché de base de datos) como en el cliente (Code Splitting, Lazy Loading, optimización de imágenes de última generación como WebP/Avif). El objetivo es ofrecer una experiencia de usuario fluida que impacte positivamente en las tasas de conversión y en el posicionamiento SEO.

## Cuándo usarla
Escenarios que activan esta skill:
- Al detectar una alta tasa de rebote en dispositivos móviles debido a tiempos de carga superiores a 3 segundos.
- Para mejorar el posicionamiento en buscadores (SEO) tras la actualización de algoritmos de Google centrados en experiencia de página.
- Durante el desarrollo de aplicaciones de gran escala (Enterprise) que sufren de "JavaScript bloat".
- Cuando se busca reducir los costes de transferencia de datos en infraestructuras cloud mediante una entrega de assets más eficiente.
- Antes de campañas de alto tráfico (Black Friday) para asegurar que el frontend no bloquee el hilo principal del navegador.

## Requisitos
- Dominio de herramientas de análisis (Lighthouse, PageSpeed Insights).
- Conocimientos de protocolos de red (HTTP/2, HTTP/3, QUIC).
- Experiencia en el manejo de CDNs (Cloudflare, Akamai).
- Familiaridad con técnicas de renderizado (SSR, SSG, ISR).
- Entendimiento del ciclo de vida del renderizado del navegador (Critical Rendering Path).
