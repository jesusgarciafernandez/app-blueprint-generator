## Descripción
Habilidad técnica para el desarrollo acelerado de aplicaciones y automatizaciones complejas sin necesidad de escribir código (No-code) o utilizando pequeñas piezas de script (Low-code). Esta skill se centra en el dominio de plataformas visuales de desarrollo (Bubble, FlutterFlow), bases de datos relacionales simplificadas (Airtable) y orquestadores de flujos (Make, Zapier, n8n). Permite la creación de MVPs funcionales, herramientas internas y automatizaciones de negocio de nivel profesional en una fracción del tiempo tradicional, manteniendo altos estándares de seguridad y escalabilidad mediante la integración de APIs y servicios cloud.

## Cuándo usarla
Escenarios que activan esta skill:
- Al requerir el lanzamiento de un Producto Mínimo Viable (MVP) en días en lugar de meses para validar una idea de negocio.
- Para automatizar procesos internos repetitivos entre diferentes herramientas SaaS (ej: CRM, Slack, Email).
- Durante la creación de paneles de administración o herramientas operativas internas personalizadas para equipos no técnicos.
- Cuando el presupuesto o los recursos de ingeniería son limitados y se requiere una solución funcional inmediata.
- Para prototipar interfaces de usuario complejas que necesiten conectividad con datos reales.

## Requisitos
- Dominio de la lógica de programación (condicionales, bucles, manejo de datos) aunque sea visualmente.
- Familiaridad con el consumo de APIs externas (REST, JSON, Webhooks).
- Conocimientos de estructuración de bases de datos relacionales.
- Suscripción y acceso a plataformas seleccionadas (ej: Bubble.io, Airtable, Make.com).
- Entendimiento de los principios básicos de UI/UX para el diseño de interfaces.
