## Descripción
Habilidad técnica avanzada para el desarrollo de aplicaciones web a gran escala utilizando TypeScript, el superset de JavaScript que añade tipado estático opcional. Esta skill se centra en el dominio del sistema de tipos (Interfaces, Types, Generics, Union/Intersection Types) para detectar errores en tiempo de desarrollo y mejorar la capacidad de refactorización del código. Abarca la configuración del compilador (`tsconfig.json`), la integración con frameworks modernos (React, Angular, Vue) y el desarrollo de backends robustos con Node.js/NestJS. El objetivo es aumentar la productividad del desarrollador, facilitar la colaboración en equipo mediante la autodocumentación del código y garantizar la calidad del software a largo plazo.

## Cuándo usarla
Escenarios que activan esta skill:
- Al desarrollar aplicaciones complejas donde la predictibilidad de los datos sea fundamental para evitar errores en producción.
- Para mejorar la experiencia de desarrollo (DX) mediante autocompletado avanzado y detección de errores inmediata en el IDE.
- Durante la refactorización de grandes bases de código JavaScript para asegurar que los cambios no rompan funcionalidades existentes.
- Cuando se trabaja en equipos multidisciplinares donde los "contratos" de datos (Interfaces) sirven como documentación técnica viva.
- Para aprovechar las últimas características de ECMAScript manteniendo la compatibilidad con navegadores antiguos mediante la transpilación.

## Requisitos
- Dominio sólido de JavaScript moderno (ES6+).
- Conocimientos de lógica de tipos estáticos.
- Instalación y configuración de herramientas de build (Vite, Webpack, SWC).
- Familiaridad con el ecosistema de definiciones de tipos (@types).
- Experiencia en el uso de IDEs preparados para TypeScript (VS Code, WebStorm).
