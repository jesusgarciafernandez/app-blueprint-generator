## Descripción
Habilidad técnica avanzada para transformar aplicaciones web tradicionales en experiencias instalables con características similares a las nativas (instalación en pantalla de inicio, modo offline, notificaciones push). Esta skill se centra en el uso de Service Workers para la gestión inteligente del caché, la configuración del Web App Manifest para la identidad visual y la implementación de estrategias de sincronización en segundo plano (Background Sync). El objetivo es maximizar la retención de usuarios mediante una carga instantánea, funcionamiento sin conexión a internet y una integración profunda con el sistema operativo del dispositivo móvil o escritorio.

## Cuándo usarla
Escenarios que activan esta skill:
- Al desarrollar aplicaciones web que deban ser accesibles en entornos con conectividad inestable o nula (ej: herramientas de almacén, viajes).
- Para evitar las comisiones y procesos de revisión de las tiendas de aplicaciones tradicionales (App Store/Google Play).
- Cuando se busca mejorar drásticamente la velocidad de carga percibida mediante el pre-cacheo de assets críticos.
- Durante la implementación de sistemas de fidelización que requieran el envío de notificaciones push directas al navegador.
- Al crear aplicaciones que se beneficien de la instalación directa sin friction de descarga.

## Requisitos
- Servidor web con HTTPS obligatorio (requisito de seguridad para Service Workers).
- Conocimientos profundos de JavaScript asíncrono (Promises, Fetch API).
- Configuración de un archivo `manifest.json` válido con iconos y colores corporativos.
- Herramientas de auditoría (ej: Lighthouse PWA).
- Frameworks de frontend modernos (React, Vue, Svelte) que faciliten la gestión del estado.
