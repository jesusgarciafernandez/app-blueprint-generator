## Descripción
Habilidad avanzada para la implementación de estrategias de renderizado en el lado del servidor (SSR) en aplicaciones web modernas. Esta skill se centra en el uso de frameworks como Next.js (React) o Nuxt.js (Vue) para generar el HTML completo en el servidor para cada petición, mejorando drásticamente el tiempo de carga inicial percibido (First Contentful Paint) y garantizando que el contenido sea plenamente indexable por los motores de búsqueda (SEO). Abarca conceptos críticos como la hidratación (Hydration), la gestión de estados compartidos entre servidor y cliente, y la optimización de caché en el borde para equilibrar el rendimiento y la frescura de los datos.

## Cuándo usarla
Escenarios que activan esta skill:
- Al desarrollar aplicaciones de comercio electrónico o blogs donde el posicionamiento orgánico (SEO) es vital para el negocio.
- Para mejorar la experiencia de usuario en dispositivos con recursos limitados o conexiones lentas, entregando HTML listo para visualizar.
- Durante la implementación de páginas con contenido altamente dinámico que debe ser personalizado por cada usuario antes de llegar al navegador.
- Cuando se busca una previsualización correcta de enlaces en redes sociales mediante meta-etiquetas dinámicas (Open Graph).
- Para reducir la dependencia de la ejecución de JavaScript en el cliente para la renderización inicial.

## Requisitos
- Dominio profundo de JavaScript/TypeScript y frameworks frontend (React, Vue).
- Conocimientos sólidos de Node.js y el ciclo de vida de las peticiones HTTP.
- Entendimiento de los principios de SEO técnico.
- Familiaridad con conceptos de hidratación y serialización de datos.
- Experiencia en el despliegues de aplicaciones Node.js y configuración de proxies (Nginx) o plataformas serverless (Vercel, Netlify).
