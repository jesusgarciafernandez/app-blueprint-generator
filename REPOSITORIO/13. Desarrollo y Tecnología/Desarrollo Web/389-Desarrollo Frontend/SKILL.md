## Descripción
Habilidad técnica y creativa centrada en la construcción de interfaces de usuario (UI) dinámicas, accesibles y de alto rendimiento para aplicaciones web. Esta skill abarca desde el dominio semántico de HTML5 y CSS3 avanzado hasta la arquitectura de aplicaciones complejas utilizando frameworks modernos como React, Vue.js o Next.js. Enseña a gestionar el estado global del cliente, optimizar la carga de activos (Critical Path Rendering), implementar diseños responsivos "Mobile-First" y asegurar la interactividad mediante JavaScript asíncrono. El objetivo es traducir diseños complejos en experiencias de usuario fluidas que funcionen perfectamente en cualquier dispositivo y navegador.

## Cuándo usarla
Escenarios que activan esta skill:
- Al desarrollar aplicaciones web de una sola página (SPA) que requieran una alta interactividad del usuario.
- Durante la creación de sistemas de diseño (Design Systems) corporativos y bibliotecas de componentes reutilizables.
- Para mejorar la puntuación de Web Vitals (LCP, FID, CLS) en productos digitales existentes.
- Al implementar interfaces que necesiten consumir datos de APIs REST o GraphQL de forma eficiente.
- Para asegurar que una plataforma web cumpla con los estándares internacionales de accesibilidad (WCAG).

## Requisitos
- Dominio de JavaScript (ES6+) y preferiblemente TypeScript.
- Entorno de desarrollo Node.js instalado con gestores de paquetes (npm/pnpm/yarn).
- Conocimientos prácticos en frameworks (React es la referencia en este repo).
- Habilidades de CSS (Sass, Tailwind CSS o CSS Modules).
- Herramientas de depuración de navegador (Chrome DevTools).
