## Descripción
Habilidad avanzada para el diseño, desarrollo y despliegue de aplicaciones web basadas en la arquitectura Jamstack (JavaScript, APIs y Markup). Esta skill se centra en la desacoplación del frontend y el backend, utilizando Generadores de Sitios Estáticos (SSG) para pre-renderizar el contenido y APIS/Functions para la lógica dinámica. Abarca el dominio de ecosistemas modernos como Gatsby, Astro o Hugo, la integración con Headless CMS (Contentful, Strapi, Sanity) y el aprovechamiento de redes de distribución global (CDNs) para garantizar una seguridad superior, un rendimiento inigualable y una escalabilidad casi infinita. El objetivo es construir experiencias web rápidas, seguras y de bajo mantenimiento operativo.

## Cuándo usarla
Escenarios que activan esta skill:
- Al desarrollar sitios corporativos, blogs o documentaciones técnicas que requieran tiempos de carga instantáneos y alta seguridad.
- Para reducir los costes de infraestructura eliminando la necesidad de gestionar servidores de bases de datos o backends tradicionales 24/7.
- Durante la creación de landing pages de alto tráfico que deban resistir picos de usuarios sin intervención manual de escalado.
- Cuando se busca una experiencia de edición modular para el equipo de contenido mediante el uso de Headless CMS.
- Para mejorar radicalmente el posicionamiento SEO mediante el servicio de archivos estáticos optimizados desde el borde (Edge).

## Requisitos
- Dominio de JavaScript (ES6+) y herramientas de build (npm, yarn, vite).
- Experiencia con al menos un Generador de Sitios Estáticos (SSG) o framework híbrido (Astro, Gatsby, 11ty).
- Conocimientos en el consumo de APIs (REST, GraphQL).
- Familiaridad con plataformas de despliegue atómico y Git-based (Netlify, Vercel, GitHub Pages).
- Entendimiento de los principios de diseño desacoplado y micro-servicios.
