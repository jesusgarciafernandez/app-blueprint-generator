## Descripción
Habilidad conceptual y técnica avanzada que aplica un paradigma de programación basado en el uso de funciones matemáticas puras y la inmutabilidad de los datos. Esta skill enseña a evitar los efectos secundarios (side-effects), gestionar el estado de forma declarativa y potenciar la reutilización de código mediante funciones de orden superior (Higher-Order Functions) como Map, Filter y Reduce. La programación funcional facilita la creación de software más predecible, fácil de testear y naturalmente preparado para la ejecución en sistemas paralelos y concurrentes.

## Cuándo usarla
Escenarios que activan esta skill:
- Al desarrollar sistemas de procesamiento de datos masivos que requieran paralelismo seguro.
- Durante la construcción de interfaces de usuario modernas (ej: React con Hooks) que se basan en la inmutabilidad estatal.
- Para reducir la complejidad y los bugs en aplicaciones con lógica de negocio altamente matemática o lógica.
- Cuando se busca maximizar la cobertura de tests unitarios mediante el uso de funciones puras fáciles de validar.
- Al implementar sistemas asíncronos complejos utilizando mónadas o flujos reactivos.

## Requisitos
- Conocimiento de lenguajes que soporten el paradigma funcional (JavaScript/TypeScript, Elixir, Haskell, Scala, Rust).
- Comprensión de conceptos matemáticos básicos (conjuntos, funciones).
- Instalación de linters que promuevan la inmutabilidad (ej: eslint-plugin-functional).
- Entorno de ejecución para pruebas rápidas (REPL).
