## Descripción
Habilidad técnica integral para la arquitectura, diseño y desarrollo de aplicaciones móviles fluidas y de alto rendimiento. Esta skill abarca los paradigmas de desarrollo nativo directo (Swift para iOS, Kotlin para Android) y soluciones multiplataforma modernas (Flutter, React Native). Se centra en la gestión del ciclo de vida de la aplicación, el consumo eficiente de recursos (batería, datos), la integración de sensores integrados (GPS, cámara, biométricos) y la sincronización recursiva de datos offline. El objetivo es entregar experiencias de usuario premium con interfaces táctiles optimizadas y una distribución segura a través de App Store y Google Play.

## Cuándo usarla
Escenarios que activan esta skill:
- Al crear aplicaciones que necesiten acceso profundo al hardware del dispositivo (biometría, acelerómetro).
- Para desarrollar plataformas B2C que requieran notificaciones push personalizadas y alta retención de usuarios.
- Durante el diseño de aplicaciones que deben funcionar sin conexión a internet (Modo Offline).
- Cuando se busca unificar el código fuente para iOS y Android sin sacrificar rendimiento nativo.
- Al implementar sistemas móviles con alta interactividad gestual y fluidez extrema (60-120 fps).

## Requisitos
- SDKs instalados (Xcode para iOS, Android Studio para Android).
- Lenguajes de programación requeridos (Swift, Kotlin, Dart o JavaScript/TypeScript).
- Entornos de emulación/simulación configurados y dispositivos físicos para testeo de performance.
- Conocimientos de patrones de diseño móviles (MVVM, Clean Architecture).
- Cuentas de desarrollador activas en Apple (Developer Program) y Google (Play Console).
