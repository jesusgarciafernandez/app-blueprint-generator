## Descripción
Habilidad de ingeniería de software avanzada que permite conectar Modelos de Lenguaje (LLMs) con fuentes de datos externas, herramientas computacionales y otros modelos para crear aplicaciones inteligentes y autónomas. La Orquestación de LLMs va mucho más allá de un simple prompt; se trata de construir arquitecturas donde el modelo puede razonar, planificar tareas, ejecutar código, consultar bases de datos (RAG) y aprender del historial de conversación. Utilizando frameworks como LangChain o LangGraph, esta skill permite transformar un modelo de lenguaje estático en un "Agente" capaz de resolver problemas complejos de principio a fin.

## Cuándo usarla
Escenarios que activan esta skill:
- Al desarrollar un asistente que debe redactar un informe basándose en 50 PDFs guardados en SharePoint (RAG avanzado).
- Para crear sistemas de "Auto-GPT" que desglosan un objetivo general en pequeñas tareas ejecutables.
- En aplicaciones multimodales que deben procesar una imagen, describirla y luego buscar productos similares en una base de datos.
- Para implementar flujos de trabajo donde varios LLMs especializados colaboran entre sí (ej: un redactor y un revisor legal).
- Al integrar LLMs con herramientas legacy vía API para automatizar procesos administrativos complejos.

## Requisitos
- Dominio de frameworks de orquestación (LangChain, LlamaIndex, Haystack).
- Comprensión de conceptos de memoria (ConversationBuffer, VectorStoreRetriever).
- Experiencia en el uso de "Tools" y "Functions" (Function Calling) de modelos como GPT-4o.
- Habilidad en el diseño de arquitecturas de Agentes y Grafos de decisión (LangGraph).
- Conocimiento de técnicas de evaluación de cadenas (LangSmith, RAGAS).
