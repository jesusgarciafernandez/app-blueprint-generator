## Descripción
Habilidad estratégica centrada en la organización sistemática y el etiquetado inteligente de la información corporativa, tanto estructurada como no estructurada. La Clasificación de Datos utiliza algoritmos de Machine Learning y NLP para identificar la naturaleza, el nivel de sensibilidad y la categoría temática de cada activo de información. Esta skill permite crear taxonomías dinámicas que facilitan la búsqueda semántica, el cumplimiento normativo (identificando PII) y la eficiencia operativa. El objetivo es transformar un "Data Swamp" en un sistema de conocimiento organizado donde cada dato tiene un rol y un nivel de acceso definido.

## Cuándo usarla
Escenarios que activan esta skill:
- Al implementar un nuevo sistema de gestión documental con miles de archivos sin etiquetas.
- Para identificar automáticamente documentos que contienen información personal sensible (PII) según GDPR.
- Durante la migración de datos hacia una base de conocimiento vectorial para un sistema RAG.
- En flujos de trabajo de atención al cliente para categorizar tickets por temática y urgencia.
- Para optimizar el almacenamiento corporativo, eliminando datos redundantes o triviales (ROT analysis).

## Requisitos
- Comprensión de arquitecturas de información y diseño de taxonomías.
- Conocimiento de técnicas de NLP (TF-IDF, Word Embeddings, Clasificación con LLMs).
- Familiaridad con normativas de privacidad de datos.
- Experiencia en el uso de herramientas de etiquetado automático y gestión de metadatos.
- Capacidad para definir esquemas jerárquicos de categorías de negocio.
