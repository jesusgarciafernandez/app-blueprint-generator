## Descripción
Esta habilidad trata a los agentes de IA no como herramientas aisladas, sino como **activos de una cartera de productividad**. El profesional diseña un "AgentFolio" donde cada agente tiene un rol, una especialidad y una misión específica. Se enfoca en la orquestación, el mantenimiento y la actualización constante de esta flota de agentes para asegurar que funcionen de manera armónica y entreguen el máximo valor operativo al negocio.

## Cuándo usarla
- **Empresas con múltiples departamentos**: Para desplegar agentes especializados en Ventas, RRHH, IT y Legal de forma coordinada.
- **Procesos de automatización masiva**: Donde las tareas requieren de una cadena de agentes que se pasan el testigo.
- **Gestión de activos digitales**: Para visualizar el retorno de inversión (ROI) de cada agente desplegado.

## Requisitos
- Repositorio centralizado de skills y misiones (ej: REPOSITORIO Antigravity).
- Orquestador de flujos (n8n, LangGraph, AutoGen).
- Framework de monitorización de rendimiento agéntico.
