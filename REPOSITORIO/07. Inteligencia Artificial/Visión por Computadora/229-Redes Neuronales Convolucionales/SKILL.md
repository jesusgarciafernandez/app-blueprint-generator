## Descripción
Habilidad de Deep Learning especializada en el diseño y entrenamiento de arquitecturas neuronales optimizadas para el procesamiento de datos con estructura de rejilla, principalmente imágenes y videos. Las Redes Neuronales Convolucionales (CNNs) emulan el funcionamiento de la corteza visual humana mediante el uso de filtros (kernels) que detectan patrones espaciales como bordes, texturas y formas complejas. Esta skill abarca desde la construcción de capas de convolución y pooling hasta el manejo de arquitecturas famosas (ResNet, EfficientNet, YOLO). El objetivo es permitir que las máquinas "vean" y entiendan el mundo visual, facilitando tareas de clasificación, detección de objetos y segmentación semántica.

## Cuándo usarla
Escenarios que activan esta skill:
- Al desarrollar un sistema de control de calidad industrial que identifica piezas defectuosas en una línea de montaje mediante cámaras.
- Para crear una aplicación de diagnóstico médico que detecta tumores en radiografías o resonancias magnéticas.
- Durante la implementación de sistemas de seguridad para detección de rostros o identificación de matrículas.
- En proyectos de agricultura inteligente para clasificar tipos de cultivos o detectar plagas mediante drones.
- Para aplicaciones de realidad aumentada que deben reconocer objetos en tiempo real.

## Requisitos
- Dominio de frameworks de Deep Learning (PyTorch o TensorFlow/Keras).
- Comprensión de operaciones de procesamiento de imágenes (Normalización, Data Augmentation).
- Conocimientos de álgebra multilineal (tensores) y cálculo para retropropagación.
- Familiaridad con arquitecturas de visión pre-entrenadas (Transfer Learning).
- Experiencia en entrenamiento con aceleración por hardware (NVIDIA GPUs/CUDA).
