## Descripción
Habilidad fundamental para el diseño, construcción y mantenimiento de la infraestructura y los sistemas que permiten recolectar, transformar y mover datos a gran escala de manera fiable. Esta skill se centra en la creación de pipelines ETL (Extract, Transform, Load) y ELT, la gestión de almacenes de datos (Data Warehouses como BigQuery, Snowflake) y lagos de datos (Data Lakes). Abarca desde el procesamiento de datos por lotes (Batch) y en tiempo real (Streaming) utilizando herramientas como Apache Spark o Kafka, hasta la orquestación de flujos de trabajo con Apache Airflow. El objetivo es convertir datos crudos en activos de información estructurados y de alta calidad para su análisis y consumo por modelos de IA.

## Cuándo usarla
Escenarios que activan esta skill:
- Al integrar múltiples fuentes de datos heterogéneas (APIs, bases de datos, archivos planos) en una única "Fuente de Verdad" corporativa.
- Para automatizar la limpieza y normalización de grandes volúmenes de datos antes de su visualización en cuadros de mando.
- Durante la construcción de sistemas de recomendación o analítica en tiempo real que requieran procesamiento de eventos de baja latencia.
- Cuando se busca optimizar el rendimiento y el coste de las consultas analíticas mediante particionamiento y modelado de datos dimensional (Star Schema).
- Para garantizar la observabilidad y el linaje de los datos en toda la cadena de procesamiento.

## Requisitos
- Dominio avanzado de SQL para transformación de datos.
- Programación en Python para desarrollo de scripts y orquestación.
- Conocimientos de arquitecturas de datos (Medallion Architecture: Bronze, Silver, Gold).
- Experiencia en herramientas de procesamiento distribuido (Spark, Pandas).
- Familiaridad con plataformas de nube (GCP, AWS, Azure) y sus servicios de datos.
