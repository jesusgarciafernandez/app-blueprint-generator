## Descripción
Habilidad técnica enfocada en la creación de ecosistemas digitales interconectados mediante plataformas de automatización No-Code y Low-Code (Zapier, Make, n8n). Esta disciplina permite eliminar tareas manuales y repetitivas, asegurando que la información fluya sin fricciones entre diferentes aplicaciones (ej: CRM, correo, gestores de tareas, hojas de cálculo). Un experto en automatización personal diseña flujos lógicos complejos, con condiciones, filtros y transformaciones de datos, permitiendo que las herramientas trabajen de forma autónoma. Esta capacidad no solo ahorra cientos de horas anuales, sino que reduce drásticamente el error humano y permite una escalabilidad de procesos que antes requeriría desarrollo de software a medida.

## Cuándo usarla
Escenarios que activan esta skill:
- Al detectar una tarea que se realiza de forma mecánica más de 3 veces por semana (ej: copiar datos de un formulario a un Excel).
- Cuando se necesita sincronizar información entre dos plataformas que no tienen integración nativa (ej: enviar menciones de Slack a Notion).
- Para la creación de sistemas de notificación inteligente que filtren solo lo crítico y lo envíen al canal adecuado (Telegram, SMS, Push).
- Durante el onboarding de clientes o empleados, automatizando el envío de accesos, correos de bienvenida y creación de registros.
- Al gestionar campañas de marketing que requieran mover leads entre diferentes bases de datos y herramientas de mailing automáticamente.

## Requisitos
- Comprensión lógica de flujos condicionales (If/Else), bucles (Iterators) y disparadores (Triggers).
- Conocimientos básicos de cómo funcionan las Webhooks y las APIs REST (JSON, Headers, Authentication).
- Dominio de al menos una plataforma líder de automatización (Zapier para simplicidad, Make para lógica visual, n8n para autohospedaje y control total).
- Habilidad básica en transformación de datos (formateo de fechas, limpieza de texto, cálculos matemáticos simples).
- Conocimiento de las políticas de seguridad y privacidad en el manejo de transferencias de datos entre aplicaciones.
