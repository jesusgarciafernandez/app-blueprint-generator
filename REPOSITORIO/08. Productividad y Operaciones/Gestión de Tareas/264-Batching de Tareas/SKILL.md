## Descripción
Habilidad operativa centrada en la agrupación de tareas de naturaleza similar en bloques de tiempo dedicados para minimizar el costo cognitivo del "cambio de contexto" (*context-switching*). Al igual que en la producción industrial, el cerebro requiere un tiempo de arranque (*setup time*) para sintonizar con un tipo específico de actividad (redacción, cálculo, reuniones, llamadas). Esta skill enseña a estructurar la agenda semanal mediante "lotes" o *batches*, permitiendo que el usuario mantenga un estado de flujo constante al no tener que reorientar su atención entre estímulos dispares, lo que resulta en una mayor velocidad de ejecución y una reducción drástica del agotamiento mental al final del día.

## Cuándo usarla
Escenarios que activan esta skill:
- Al gestionar múltiples canales de comunicación (emails, Slack, correos) que fragmentan el día.
- Durante la fase de planificación semanal para organizar tareas administrativas menores.
- En procesos creativos que requieren fases diferenciadas (ej: investigación, redacción, edición).
- Para optimizar tareas logísticas o físicas (pagos de facturas, recados, llamadas telefónicas).
- Al notar que se salta constantemente de una tarea a otra sin llegar a terminar ninguna adecuadamente.

## Requisitos
- Sistema de gestión de tareas que permita el etiquetado por "contexto" o "tipo de actividad".
- Control sobre la agenda (Time Blocking) para reservar bloques exclusivos.
- Disciplina para no romper el bloque con tareas que no pertenecen al lote actual.
- Capacidad para prever la duración total de un lote de tareas similares.
- Herramientas de automatización para pre-clasificar tareas (opcional).
