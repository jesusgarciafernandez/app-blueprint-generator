## Descripción
Habilidad avanzada en el uso y configuración de ecosistemas de gestión de tareas y proyectos, adaptando las herramientas a las necesidades específicas de cada flujo de trabajo. Esta disciplina abarca desde la gestión de listas de tareas individuales de alto rendimiento (Todoist, Things) hasta la administración de complejos entornos de desarrollo ágil y gestión de tickets empresariales (Jira, Linear, ClickUp). Un experto en esta skill no solo "apunta cosas", sino que estructura la jerarquía de la información, define estados de flujo de trabajo personalizados, implementa metodologías Kanban o Scrum, y asegura que la carga de trabajo sea visible, priorizada y ejecutable de forma realista para él mismo o su equipo.

## Cuándo usarla
Escenarios que activan esta skill:
- Al iniciar un nuevo proyecto que involucre a múltiples colaboradores y requiera un seguimiento detallado de hitos y dependencias.
- Cuando la gestión de pendientes individuales se vuelve caótica y se necesita una herramienta con filtros, etiquetas y recordatorios inteligentes.
- En el desarrollo de software o productos digitales que sigan metodologías Ágiles (Sprints, Backlog, Burndown charts).
- Para la centralización de incidencias, errores (*bugs*) o peticiones de clientes que necesiten una resolución estructurada y trazable.
- Al gestionar equipos remotos donde la visibilidad del estado de las tareas sea crítica para evitar reuniones de estatus innecesarias.

## Requisitos
- Dominio funcional de herramientas de gestión (Todoist para personal/ligero, Jira/Linear para product/tech).
- Entendimiento de metodologías de priorización (Matriz de Eisenhower, Método ABCDE, MoSCoW).
- Conocimiento de los principios de metodologías Ágiles (Kanban, Scrum, Lean).
- Capacidad para configurar automatizaciones nativas dentro de las plataformas (ej: Jira Automation, Todoist filters).
- Disciplina para el mantenimiento diario de las listas (limpieza, actualización de fechas y estados).
