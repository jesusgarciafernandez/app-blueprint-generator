## Descripción
Habilidad de Gestión del Conocimiento Personal (PKM) centrada en la captura estructurada, curación y recuperación rápida de referencias digitales (URLs, artículos, papers, recursos multimedia). Más allá del simple almacenamiento de "enlaces favoritos", esta skill enseña a construir un "Segundo Cerebro" (Building a Second Brain) mediante la categorización semántica, el etiquetado funcional y la síntesis de la información guardada. El objetivo no es la acumulación de datos (*hoarding*), sino la creación de una biblioteca dinámica que sirva como motor de consulta para proyectos activos, aprendizaje continuo y generación de nuevas ideas.

## Cuándo usarla
Escenarios que activan esta skill:
- Durante procesos de investigación para un nuevo proyecto, artículo o desarrollo técnico.
- Al encontrar recursos valiosos que no pueden ser consumidos en el momento (*Read-it-later*).
- Para organizar herramientas, tutoriales y plantillas que se utilizan de forma recurrente.
- Cuando la cantidad de pestañas abiertas en el navegador o marcadores desordenados impide la fluidez operativa.
- Al construir un repositorio de inspiración (*Swipe File*) para procesos creativos de marketing o diseño.

## Requisitos
- Herramientas de captura y gestión (ej: Raindrop.io, Pocket, Instapaper, MyMind o bases de datos en Notion/Obsidian).
- Extensiones de navegador para el web-clipping (captura de contenido).
- Un sistema de taxonomía simple pero robusto (ej: Método PARA de Tiago Forte: Proyectos, Áreas, Recursos, Archivos).
- Disciplina para la revisión periódica del contenido guardado.
- Capacidad para discernir entre información de alto valor y "ruido" digital.
