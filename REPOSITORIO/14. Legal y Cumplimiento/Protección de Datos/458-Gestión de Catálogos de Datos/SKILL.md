## Descripción
Habilidad fundamental de Gobernanza de Datos que consiste en la creación y mantenimiento de un inventario centralizado y organizado de todos los activos de información de una organización. La Gestión de Catálogos de Datos actúa como el "Google interno" de los datos, permitiendo a los usuarios (científicos de datos, analistas, auditores) descubrir qué datos existen, de dónde vienen (linaje), quién es el dueño, qué tan confiables son y si contienen información sensible. Esta skill es el puente entre los silos tecnológicos y el uso democrático y seguro de la información corporativa.

## Cuándo usarla
Escenarios que activan esta skill:
- En grandes corporaciones donde los equipos no saben si una base de datos ya existe antes de intentar crear una nueva.
- Durante auditorías de cumplimiento (RGPD) para identificar rápidamente dónde se almacena la información personal de los clientes.
- Para mejorar la calidad de los datos, etiquetando activos como "certificados" o "en proceso de limpieza".
- Al integrar nuevas fuentes de datos provenientes de fusiones o adquisiciones de empresas.
- Para facilitar el "onboarding" de nuevos analistas, dándoles un mapa claro del ecosistema de datos de la empresa.

## Requisitos
- Conocimiento de estándares de metadatos (Dublin Core, ISO/IEC 11179).
- Dominio de herramientas de catálogo (Atlan, Collibra, Apache Atlas, Google Data Catalog).
- Comprensión de conceptos de Linaje de Datos (Data Lineage) y Glosario de Negocio.
- Capacidad para clasificar datos según su sensibilidad (Público, Confidencial, Privado).
- Habilidad pedagógica para evangelizar a los equipos en la importancia de documentar sus datos.
