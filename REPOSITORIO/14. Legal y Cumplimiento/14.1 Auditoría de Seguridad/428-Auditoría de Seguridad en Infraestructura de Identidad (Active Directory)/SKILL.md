## Descripción
Esta habilidad se enfoca en proteger el "corazón" de la red corporativa: el sistema de gestión de identidades. El profesional audita de forma proactiva las configuraciones de Active Directory (AD) para detectar vulnerabilidades comunes, configuraciones erróneas y vectores de ataque que podrían permitir a un atacante comprometer la totalidad del dominio.

## Cuándo usarla
- **Auditorías de cumplimiento anuales**: Para asegurar que los controles de acceso cumplen con normativas como ISO 27001 o Esquema Nacional de Seguridad (ENS).
- **Tras cambios mayores en la arquitectura de red**: Para validar que no se han introducido brechas de seguridad involuntarias.
- **Respuesta a incidentes**: Para investigar posibles compromisos de cuentas y reforzar las defensas post-ataque.

## Requisitos
- Conocimiento profundo de protocolos de red (Kerberos, NTLM, LDAP).
- Herramientas de auditoría (ej: BloodHound, PingCastle, SharpHound).
- Privilegios de administrador o auditor para acceder a la configuración del dominio.
