# 🛸 Guía de Usuario: Antigravity Skill Explorer v1.1

Bienvenido al ecosistema **Antigravity**, un repositorio maestro de conocimiento técnico diseñado para ser consultado y operado por humanos y agentes de IA. Esta guía te explicará cómo sacar el máximo provecho a este sistema.

---

## 📂 Estructura del Repositorio

El proyecto está organizado en **20 Áreas de Conocimiento** principales. Cada una contiene subcategorías y unidades de conocimiento individuales (Skills).

- **Carpeta Raíz**: Contiene los archivos del sistema y las carpetas de categorías (ej: `01. Marketing Digital`).
- **Nivel de Skill**: Cada skill reside en su propia carpeta y contiene un archivo `SKILL.md` con el manual operativo detallado.

---

## 🔍 Cómo Explorar las Skills

Para visualizar y buscar entre las más de 478 habilidades de forma atractiva y rápida:

1. Localiza el archivo **`EXPLORADOR.html`** en la raíz.
2. Haz doble clic para abrirlo en tu navegador favorito.
3. Utiliza la **barra de búsqueda** para encontrar términos específicos o navega por las **categorías del menú lateral**.
4. Haz clic en **"Consultar Manual"** para abrir el archivo técnico detallado.

---

## ⚡ Sistemas de Sincronización

Para que el explorador se actualice solo al añadir o modificar Skills, debes activar el sistema de vigilancia.

### 🍎 En macOS (Recomendado)
- Haz doble clic en `VIGILANTE_ANTIGRAVITY.command`.
- Verás notificaciones nativas cada vez que guardes un archivo `SKILL.md`.

### 🪟 En Windows
- Asegúrate de tener instalado [Python 3](https://www.python.org/).
- Haz doble clic en `VIGILANTE_WINDOWS.bat`. 
- Una ventana negra se abrirá y monitorizará tus cambios.

### 🐧 En Linux
- Abre una terminal en la carpeta del proyecto.
- Ejecuta `./VIGILANTE_LINUX.sh` (asegúrate de darle permisos con `chmod +x VIGILANTE_LINUX.sh`).
- Requiere `python3` y `libnotify` para las alertas.
