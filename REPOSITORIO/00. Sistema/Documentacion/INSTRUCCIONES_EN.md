# 🛸 User Guide: Antigravity Skill Explorer v1.1

Welcome to the **Antigravity** ecosystem, a master repository of technical knowledge designed to be consulted and operated by both humans and AI agents. This guide will explain how to get the most out of this system.

---

## 📂 Repository Structure

The project is organized into **20 main Areas of Knowledge**. Each contains subcategories and individual knowledge units (Skills).

- **Root Folder**: Contains system files and category folders (e.g., `01. Digital Marketing`).
- **Skill Level**: Each skill resides in its own folder and contains a `SKILL.md` file with a detailed operational manual.

---

## 🔍 How to Explore Skills

To visualize and search among the 478+ skills in an attractive and fast way:

1. Locate the **`EXPLORADOR.html`** file in the root directory.
2. Double-click to open it in your favorite browser.
3. Use the **search bar** to find specific terms or navigate through the **sidebar categories**.
4. Click on **"Open Manual"** to open the detailed technical file.

---

## ⚡ Synchronization Systems

To keep the explorer updated automatically when adding or modifying Skills, you must activate the monitoring system.

### 🍎 On macOS (Recommended)
- Double-click on `VIGILANTE_ANTIGRAVITY.command`.
- You will see native notifications every time you save a `SKILL.md` file.

### 🪟 On Windows
- Ensure you have [Python 3](https://www.python.org/) installed.
- Double-click on `VIGILANTE_WINDOWS.bat`. 
- A terminal window will open and monitor your changes.

### 🐧 On Linux
- Open a terminal in the project folder.
- Run `./VIGILANTE_LINUX.sh` (ensure it has permissions with `chmod +x VIGILANTE_LINUX.sh`).
- Requires `python3` and `libnotify` for alerts.

---

## 🏗️ Technical Structure
The project is designed to be lightweight and portable:
- **`EXPLORADOR.html`**: Principal user interface.
- **`INDEX.js`**: Automatically generated dynamic database.
- **`00. Sistema/`**: Contains the indexing engine (`ACTUALIZAR.py`) and the watcher (`WATCH.py`).
- **`INSTRUCCIONES.md`**: This user manual.
