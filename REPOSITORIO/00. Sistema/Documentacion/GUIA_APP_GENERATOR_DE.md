# 🛸 Leitfaden für den App Blueprint Generator

## Was ist der App Blueprint Generator?

Es ist ein Werkzeug, das Ihre **Anwendungsidee** (mit eigenen Worten beschrieben) in ein **professionelles Paket** verwandelt, das bereit ist, von jeder Künstlichen Intelligenz für Sie aufgebaut zu werden.

Sie müssen nicht programmieren können. Sie müssen nur wissen, was Sie wollen.

---

## Anforderungen

- **Python 3** auf Ihrem Computer installiert ([hier herunterladen](https://www.python.org/))
- **Webbrowser** (Chrome, Firefox, Safari oder Edge)
- Dieses komplette Repository auf Ihrem Gerät heruntergeladen

---

## Verwendung (Schritt für Schritt)

### 1. Generatoren starten

Führen Sie einen **Doppelklick** auf die Datei aus, die Ihrem Betriebssystem entspricht:

| System | Datei |
|---------|---------|
| macOS | `CREAR_APP.command` |
| Windows | `CREAR_APP.bat` |
| Linux | `CREAR_APP.sh` |

Ihr Browser öffnet sich automatisch mit der Benutzeroberfläche des Generators.

### 2. Idee beschreiben (5 einfache Schritte)

1. **Vision**: Geben Sie Ihrer App einen Namen und beschreiben Sie, was sie tut.
2. **Zielgruppe**: Geben Sie an, wer sie in welchem Bereich nutzen wird.
3. **Funktionen**: Wählen Sie die Funktionen aus, die Sie benötigen.
4. **Design**: Wählen Sie Farben und visuellen Stil.
5. **Generieren**: Überprüfen Sie die Zusammenfassung und klicken Sie auf „Generate My Blueprint“.

### 3. Blueprint herunterladen

Das System generiert eine ZIP-Datei, die automatisch im Ordner **`APPS-CREADAS`** innerhalb des Repositories gespeichert wird.

### 4. KI öffnen und mit dem Aufbau beginnen

1. Öffnen Sie Ihr bevorzugtes KI-Tool (Antigravity, Cursor, Claude, ChatGPT...)
2. Öffnen Sie die Datei `LEEME_PRIMERO.md` aus der heruntergeladenen ZIP-Datei.
3. **Kopieren Sie den gesamten Text** und fügen Sie ihn als erste Nachricht ein.
4. Die KI beginnt mit dem Aufbau Ihrer Anwendung!

## Fragen und Antworten (FAQ)

### Benötige ich eine Internetverbindung?
**Nein.** Alles läuft lokal. Internet wird nur zum Laden der Benutzeroberflächen-Schriftarten (Google Fonts) benötigt, es funktioniert aber auch ohne sie.

### Kann ich mehr als einen Blueprint erstellen?
**Ja.** Sie können beliebig viele erstellen. Jeder wird mit eigenem Namen und Datum gespeichert.

---

## Fehlerbehebung

| Problem | Lösung |
|----------|----------|
| „Server offline“ | Schließen Sie die Datei `CREAR_APP.*` und öffnen Sie sie erneut |
| Browser öffnet sich nicht | Öffnen Sie manuell `http://localhost:5500` |
| Python-Fehler | Stellen Sie sicher, dass Python 3 installiert ist: `python3 --version` |
