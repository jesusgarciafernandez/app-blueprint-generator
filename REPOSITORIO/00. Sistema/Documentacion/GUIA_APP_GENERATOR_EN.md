# 🛸 App Blueprint Generator Guide

## What is the App Blueprint Generator?

It is a tool that transforms your **application idea** (described in your own words) into a **professional package** ready for any Artificial Intelligence to build the app for you.

You don't need to know how to code. You just need to know what you want.

---

## Requirements

- **Python 3** installed on your computer ([download here](https://www.python.org/))
- **Web browser** (Chrome, Firefox, Safari, or Edge)
- This complete repository downloaded to your machine

---

## How to use it (Step by Step)

### 1. Start the generator

**Double-click** the file corresponding to your operating system:

| System | File |
|---------|---------|
| macOS | `CREAR_APP.command` |
| Windows | `CREAR_APP.bat` |
| Linux | `CREAR_APP.sh` |

Your browser will automatically open with the generator interface.

### 2. Describe your idea (5 simple steps)

1. **Vision**: Give your app a name and describe what it does.
2. **Audience**: Indicate who will use it and in which sector.
3. **Features**: Select the functionalities you need.
4. **Design**: Choose colors and visual style.
5. **Generate**: Review the summary and press "Generate My Blueprint".

### 3. Download your Blueprint

The system will generate a ZIP file that will be automatically saved in the **`APPS-CREADAS`** folder within the repository.

### 4. Open your AI and start building

1. Open your favorite AI tool (Antigravity, Cursor, Claude, ChatGPT...)
2. Open the `LEEME_PRIMERO.md` (or `READ_ME_FIRST.md`) file from the downloaded ZIP.
3. **Copy all the text** and paste it as your first message.
4. The AI will start building your application!

## Frequently Asked Questions

### Do I need an internet connection?
**No.** Everything runs locally. You only need the internet to load interface fonts (Google Fonts), but it works just as well without them.

### Can I create more than one Blueprint?
**Yes.** You can create as many as you want. Each one is saved with its own name and date.

### Can I modify the Blueprint after creating it?
**Yes.** The files in the ZIP are Markdown (.md) documents that you can open and edit with any text editor.

### Does it work with any AI?
**Yes.** The Blueprint is designed to work with Antigravity, Cursor, Claude, ChatGPT, and any tool that understands natural language.

## Troubleshooting

| Problem | Solution |
|----------|----------|
| "Server offline" | Close and reopen the `CREAR_APP.*` file |
| Browser doesn't open | Manually open `http://localhost:5500` |
| Python Error | Ensure you have Python 3 installed: `python3 --version` |
| ZIP is empty | Check that the `INDEX.js` file exists in the root |
