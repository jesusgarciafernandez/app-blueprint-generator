# 🛸 Guide d'utilisation : Antigravity Skill Explorer v1.1

Bienvenue dans l'écosystème **Antigravity**, un répertoire maître de connaissances techniques conçu pour être consulté et utilisé par les humains et les agents d'IA. Ce guide vous expliquera comment tirer le meilleur parti de ce système.

---

## 📂 Structure du Répertoire

Le projet est organisé en **20 Domaines de Connaissances** principaux. Chacun contient des sous-catégories et des unités de connaissances individuelles (Skills).

- **Dossier Racine** : Contient les fichiers du système et les dossiers de catégories (ex: `01. Marketing Numérique`).
- **Niveau de Skill** : Chaque skill réside dans son propre dossier et contient un fichier `SKILL.md` avec un manuel opérationnel détaillé.

---

## 🔍 Comment Explorer les Skills

Pour visualiser et rechercher parmi les 478+ compétences de manière attrayante et rapide :

1. Localisez le fichier **`EXPLORADOR.html`** à la racine.
2. Double-cliquez pour l'ouvrir dans votre navigateur préféré.
3. Utilisez la **barre de recherche** pour trouver des termes spécifiques ou naviguez dans les **catégories du menu latéral**.
4. Cliquez sur **"Consulter le Manuel"** pour ouvrir le fichier technique détaillé.

---

## ⚡ Systèmes de Synchronisation

Pour que l'explorateur se mette à jour automatiquement lors de l'ajout ou de la modification de Skills, vous devez activer le système de surveillance.

---

## 🏗️ Structure Technique
Le projet est conçu pour être léger et portable :
- **`EXPLORADOR.html`** : Interface utilisateur principale.
- **`INDEX.js`** : Base de données dynamique générée automatiquement.
- **`00. Sistema/`** : Contient le moteur d'indexation (`ACTUALIZAR.py`) et le surveillant (`WATCH.py`).
- **`INSTRUCCIONES.md`** : Ce manuel d'utilisation.
