# 🛸 App Blueprint Generator ガイド

## App Blueprint Generator とは何ですか？

あなたの**アプリのアイデア**（あなた自身の言葉で説明）を、人工知能があなたの代わりにアプリを構築できる状態の**プロフェッショナルなパッケージ**に変換するツールです。

プログラミングの知識は必要ありません。何を作りたいかを知っているだけで十分です。

---

## 要件

- お使いのコンピュータに **Python 3** がインストールされていること ([ここからダウンロード](https://www.python.org/))
- **Web ブラウザ** (Chrome, Firefox, Safari, Edge)
- このリポジトリ全体をお使いの機器にダウンロード済みであること

---

## 使い方 (ステップバイステップ)

### 1. ジェネレーターを起動する

お使いの OS に対応するファイルを**ダブルクリック**します：

| システム | ファイル |
|---------|---------|
| macOS | `CREAR_APP.command` |
| Windows | `CREAR_APP.bat` |
| Linux | `CREAR_APP.sh` |

ブラウザが自動的に開き、ジェネレーターのインターフェースが表示されます。

### 2. アイデアを説明する (5 つの簡単なステップ)

1. **ビジョン**: アプリに名前を付け、その機能を説明します。
2. **対象**: 誰が、どの分野でそれを使用するかを指定します。
3. **機能**: 必要な機能を選択します。
4. **デザイン**: 色とビジュアルスタイルを選択します。
5. **生成**: 概要を確認し、"Generate My Blueprint" を押します。

### 3. ブループリントをダウンロードする

システムが ZIP ファイルを生成し、リポジトリ内の **`APPS-CREADAS`** フォルダに自動的に保存されます。

### 4. AI を開いて構築を開始する

1. お気に入りの AI ツール (Antigravity, Cursor, Claude, ChatGPT...) を開きます。
2. ダウンロードした ZIP ファイル内の `LEEME_PRIMERO.md` ファイルを開きます。
3. **すべてのテキストをコピー**して、最初のメッセージとして貼り付けます。
4. AI があなたのアプリケーションの構築を開始します！

## よくある質問 (FAQ)

### インターネット接続は必要ですか？
**いいえ。** すべてローカルで動作します。インターフェースのフォント (Google Fonts) を読み込むためにのみインターネットが必要ですが、インターネットがなくても同様に動作します。
