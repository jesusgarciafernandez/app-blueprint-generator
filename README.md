## 🌟 La Magia del Repositorio Transversal

A diferencia de otros generadores que se basan solo en prompteos planos, esta herramienta cuenta con una bóveda interna de **más de 500 Skills profesionales operativas**.

Utilizando un motor de procesamiento natural propio **(100% Zero Dependencies)**, escanea estas capacidades técnicas y arquitectónicas, extrayendo el conocimiento estructurado exacto —desde Arquitectura de Bases de Datos hasta límites de Memoria en Sistemas Multi-Agente— e inyectándolo dinámicamente como conocimiento maestro al LLM constructor final.

### Características Clave
- 🤖 **Agnóstico y Dinámico:** Prepárate para exportar contextos enteros en XML (optimizado para Claude) o directamente en metadatos `.cursorrules` / `.windsurfrules`.
- 🧠 **Niveles de Inteligencia Variables (MAS):** Tú decides si quieres compilar directrices ligeras para apps estándar, o integrar una super-conciencia de Sistema Multi-Agente (Arquitecturas con Memoria Episódica, Límites de Tokens y Control Analítico) en un click.
- 🌍 **Soporte Global:** Interfaz gráfica nativa en 6 idiomas —Español, Inglés, Francés, Alemán, Chino Mandarín y Japonés— operando 100% *Offline-Ready*.
- ⚡ **Desempeño Autocontenido:** No requiere gestores dependientes (NPM) ni internet. Ejecútalo nativamente para resguardar tus variables y secretos con máxima privacidad.
- 👤 **Perspectiva "Human-Centric":** Creado en marco a un diseño que empodera al ideador humano controlando a la máquina. (Ver [HUMAN_CENTRIC_AI.md](HUMAN_CENTRIC_AI.md)).

## 🚀 Inicio Rápido (Quickstart)

### Pre-requisitos
- Contar con la estructura de este Repositorio.
- Tener **Python 3.6+** (Instalado por defecto en macOS y Linux). ¡0 requerimientos extras!

### Ejecutar App Blueprint Generator
Ingresa a este directorio madre y arranca el script correspondiente a tu Sistema Operativo:

**Para Mac / OS / Linux**
```bash
./CREAR_APP.sh

# Script interactivo macOS:
./CREAR_APP.command
```

**Para Windows**
```cmd
CREAR_APP.bat
```

Un simple clic iniciará el gestor y tu navegador por defecto te redirigirá a la interfaz en local (http://localhost:5500).

## 🛠️ Cómo Funciona
1. **Configuración Inicial:** Describe tu negocio y selecciona qué "Intelecto" (Clásico, Copiloto o Ecosistema MAS) deseas otorgarle a tu aplicación.
2. **Conexiones Semánticas:** El algoritmo escanea la Bóveda de Skills integrando automáticamente las dependencias lógicas.
3. **Descarga de Empaquetado:** Recibe el blueprint agnóstico optimizado específicamente para tu IDE objetivo (Cursor, Claude...) y comienza a generar código de calidad empresarial.
